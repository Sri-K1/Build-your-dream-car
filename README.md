# Build-your-dream-car
A website to build your dream spec cars choose your favorite type of car, favorite color, rim, and spoiler for your dream car! (will be adding more customizations and cars soon). Explore this refined and user-friendly website to build cars from BMW to Lamborghini to Mercedes, more cars to be added soon. This website is inspired by major car brands' car customizing/ordering website. Picture credits for this website: Google images, BMW, Lamborghini, Mercedes. Sorry if the images are blurry, I tried fixing it but couldn't. 

You can build your car by clicking this link: https://dreamgaragebuildyourdreams.netlify.app/ 

# Tools/Languages used
- HTMl
- CSS
- Javascript
- VS Code
- AI (for learning purposes and debugging issues with code and help with favicon creation, etc.)
- Image cleanup and AI image upscaler
- Cubic-bezier.com (for the smooth transitions)
- cssgradient.io for gradient creation as its much easier to make on this website

# Background
So the main reason for me making this website is that I love cars and I've been a huge car guy since I was a kid. I really like how car brands like BMW, Mercedes, Tesla, etc. have a really cool website for customers to customize, build, and order their very own car and I took that as a inspiration to make my own but using car models from BMW, Mercedes, and Lamborghini.
